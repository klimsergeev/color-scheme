# Price Color Mapper

Инструмент для визуализации распределения цен билетов на схемах залов с использованием цветового кодирования.

## Описание

Price Color Mapper предоставляет два алгоритма для маппинга цен на цветовые шкалы:

1. **d3-scale (квантильный)** - использует библиотеку D3.js для создания квантильных шкал цветов на основе распределения цен
2. **CDF (кумулятивная функция распределения)** - алгоритм на основе кумулятивной функции распределения для более точного отображения ценовых диапазонов

## Как запустить

Проект требует локального веб-сервера из-за ограничений CORS при загрузке SVG файлов.

### Вариант 1: Python

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

### Вариант 2: Node.js (http-server)

```bash
npx http-server -p 8000
```

### Вариант 3: PHP

```bash
php -S localhost:8000
```

После запуска сервера откройте в браузере:

- **D3-scale версия**: http://localhost:8000/d3-scale-update/
- **CDF версия**: http://localhost:8000/cdf-update/

## Использование

1. Выберите схему зала из выпадающего списка
2. Схема загрузится с цветовым кодированием цен
3. Наведите курсор на места, чтобы увидеть информацию о цене

## Технологии

- Vanilla JavaScript
- D3.js (для d3-scale версии)
- SVG
